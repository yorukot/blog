# 這個網站的 Markdown 語法完全指南

歡迎來到我的部落格！這篇文章會詳細介紹這個網站支援的所有 Markdown 語法，包括一些我特別設計的酷炫功能。

## 基礎 Markdown 語法

### 標題系統

這個網站支援完整的標題層級，從 `#` 到 `######`：

```markdown
# 一級標題 (H1)
## 二級標題 (H2) 
### 三級標題 (H3)
#### 四級標題 (H4)
##### 五級標題 (H5)
###### 六級標題 (H6)
```

特別的是，二級標題會有漂亮的底線裝飾，讓你的文章更有層次感！

### 文字格式化

```markdown
**粗體文字** 或 __粗體文字__
*斜體文字* 或 _斜體文字_
~~刪除線~~
`行內程式碼`
```

### 連結和圖片

```markdown
[連結文字](https://example.com)
![圖片描述](圖片網址)
```

圖片會有漂亮的圓角和陰影效果，滑鼠懸停時還會稍微放大！

### 列表

**無序列表：**
```markdown
- 項目 1
- 項目 2
  - 子項目 2.1
  - 子項目 2.2
- 項目 3
```

**有序列表：**
```markdown
1. 第一項
2. 第二項
3. 第三項
```

### 引用區塊

```markdown
> 這是一個引用區塊
> 可以有多行
> 
> 也可以包含**粗體**和`程式碼`
```

引用區塊會有左邊的彩色邊框和背景色，看起來很專業！

### 程式碼區塊

**行內程式碼：**
```markdown
使用 `console.log('Hello World')` 來輸出訊息
```

**程式碼區塊：**
````markdown
```javascript
function hello() {
  console.log('Hello World!');
}
```
````

程式碼區塊有語法高亮、複製按鈕，還有漂亮的圓角設計！

### 表格

```markdown
| 欄位 1 | 欄位 2 | 欄位 3 |
|--------|--------|--------|
| 內容 1 | 內容 2 | 內容 3 |
| 內容 4 | 內容 5 | 內容 6 |
```
| 欄位 1 | 欄位 2 | 欄位 3 |
|--------|--------|--------|
| 內容 1 | 內容 2 | 內容 3 |
| 內容 4 | 內容 5 | 內容 6 |
表格有漂亮的樣式和懸停效果！

### 分隔線

```markdown
---
```

## 自訂功能

### 1. 警告框 (Alert) 組件

這是我最喜歡的功能之一！你可以創建不同類型的警告框：

<Alert type="tip" title="小提示">
這是一個提示框，用來分享有用的資訊！
</Alert>

<Alert type="warning">
這是一個警告框，用來提醒重要事項。
</Alert>

<Alert type="error">
這是一個錯誤框，用來顯示錯誤訊息。
</Alert>

<Alert type="info">
這是一個資訊框，用來提供一般資訊。
</Alert>

<Alert type="success">
這是一個成功框，用來慶祝成功！
</Alert>

**使用方法：**
```markdown
<Alert type="tip" title="自訂標題">
這裡是你的內容
</Alert>
```

支援的類型：
- `tip` - 提示 (燈泡圖示)
- `warning` - 警告 (三角形圖示)  
- `error` - 錯誤 (X 圖示)
- `info` - 資訊 (i 圖示)
- `success` - 成功 (勾選圖示)

### 2. 程式碼標籤頁 (CodeTabs)

當你需要展示多種語言的程式碼時，這個功能超好用：

<CodeTabs tabs={['JavaScript', 'Python', 'Go']}>
```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

```python
def greet(name):
    return f"Hello, {name}!"
```

```go
func greet(name string) string {
    return fmt.Sprintf("Hello, %s!", name)
}
```
</CodeTabs>

### 3. 任務清單

```markdown
- [x] 完成基礎功能
- [ ] 添加進階功能
- [ ] 測試所有功能
- [ ] 部署到生產環境
```

### 4. 鍵盤按鍵

```markdown
按 <kbd>Ctrl</kbd> + <kbd>C</kbd> 來複製
按 <kbd>Ctrl</kbd> + <kbd>V</kbd> 來貼上
```

### 5. 高亮文字

```markdown
這是一段包含<mark>高亮</mark>的文字
```

### 6. 可摺疊內容

<details>
<summary>點擊展開更多內容</summary>

這裡是可以摺疊的內容，適合放置額外的資訊或詳細說明。

- 可以包含列表
- 可以包含**粗體**
- 可以包含`程式碼`

</details>

**使用方法：**
```markdown
<details>
<summary>標題</summary>

內容...
</details>
```
